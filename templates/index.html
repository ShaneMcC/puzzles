Puzzle Checklist

⬇ Tools ⬇
  1. Check the HTML Source for puzzle
  2. Does the title of the puzzle help at all?
  3. If multiples, is there anything in common? Try searching
  4. Can it be anagrammed?
  5. First/Later letters of each thing spell anything
  6. Can the elements be split into groups?
  7. Can the groups be morse/binary encoded? Check 1st, 9th, 17th etc for 0/1/something in common
  8. Can this be converted to or from braille?
  9. Is this some kind of ASCII art for a word?
  10. Can the picture be a simple word
  11. If there are multiple pictures, is there a theme commonality?
  12. Check Exif/Size
  13. Could it be a flag? Enter some search terms to find flags
  14. Run through strings/file, look at results
  15. Check file names
  16. What times do events happen or how long do they take
  17. Check file names
  18. Run through strings/file, look at results
  19. Check channels match
  20. Reverse
